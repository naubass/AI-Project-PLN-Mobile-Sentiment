Kelas

Support Vector Machine (SVM) adalah algoritma machine learning yang bekerja dengan mencari hyperplane terbaik untuk memisahkan kelas data. Pada sistem ini, SVM dilatih dengan data balanced untuk mendeteksi sentimen ulasan PLN Mobile secara akurat.

1

Cleaning: Menghapus mention, hashtag, URL, angka, dan tanda baca.

2

Tokenizing: Memecah teks menjadi satuan kata (token).

3

Vectorizing: TF-IDF mengubah token menjadi representasi numerik.

Confidence score dihasilkan dari decision function SVM yang menunjukkan seberapa jauh data dari hyperplane. Nilai dinormalisasi ke rentang 0–100%. Semakin tinggi nilainya, semakin yakin model terhadap prediksinya.
